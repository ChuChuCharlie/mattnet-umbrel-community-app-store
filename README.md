## Umbrel Mattnet App Store

This is a custom app store for UmbrelOS. Apps here are likely buggy or require further configuration after install. [Official Umbrel App Store](https://github.com/getumbrel/umbrel-apps).

## How to use:

Add GitHub url (https://github.com/ChuChuCharlie/mattnet-umbrel-community-app-store)to the umbrelOS user interface as shown in the following demo:
